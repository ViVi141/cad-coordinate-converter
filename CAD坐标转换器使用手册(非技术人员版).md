# CAD坐标转换器使用手册 v1.3.0

## 📖 快速开始

### 第一步：安装程序
1. 确保您的电脑已安装Python 3.7或更高版本
2. 双击运行 `安装依赖.bat` 文件
3. 等待安装完成

### 第二步：启动程序
1. 双击 `CAD坐标转换器.py` 文件
2. 程序会自动启动并显示主界面

## 🎯 基本操作

### 1. 选择坐标文件
- 点击"浏览"按钮
- 选择包含坐标数据的TXT文件
- 程序会自动检测文件编码和格式

### 2. 选择转换类型
- **多段线 (PLINE)**: 生成连续的多段线，适合绘制封闭图形
- **直线 (LINE)**: 生成独立的直线段，适合绘制开放图形
- **点 (POINT)**: 生成独立的点，适合标记位置

### 3. 设置高级选项
- **添加文字标注**: 为每个坐标点添加文字标注
- **文字高度**: 设置文字标注的高度
- **按分组分别处理**: 如果文件包含分组，可以分别处理每个组
- **转换后自动复制**: 转换完成后自动复制到剪贴板

### 4. 开始转换
- 点击"开始转换"按钮
- 等待转换完成
- 查看转换结果

## 📁 支持的坐标格式

### 标准格式
```
X坐标, Y坐标
X坐标, Y坐标, Z坐标
```

### 其他格式
```
X坐标 Y坐标 Z坐标
X坐标	Y坐标	Z坐标
X坐标; Y坐标; Z坐标
X坐标， Y坐标， Z坐标
```

### 分组格式
```
第1组
坐标1, 坐标2
坐标3, 坐标4

第2组
坐标5, 坐标6
坐标7, 坐标8
```

## 📋 文件要求

### 文件格式
- 文件类型：TXT文本文件
- 编码格式：UTF-8、GBK、GB2312等（程序自动检测）
- 文件大小：建议不超过50MB

### 坐标数据
- 坐标值：数字格式，支持小数
- 分隔符：逗号、空格、制表符、分号
- 精度：保持原始精度，不进行强制格式化

## 🎨 界面说明

### 左侧控制面板
- **文件选择**: 选择要转换的坐标文件
- **转换选项**: 设置转换类型和参数
- **高级设置**: 配置文字标注和分组处理
- **操作按钮**: 执行转换、复制、保存等操作

### 右侧结果显示
- **CAD命令**: 显示生成的CAD命令
- **数据预览**: 显示原始文件内容
- **图形预览**: 显示坐标图形（需要matplotlib）

## 🔧 高级功能

### 分组处理
如果您的坐标文件包含分组信息，可以：
1. 勾选"按分组分别处理"
2. 程序会为每个分组生成独立的CAD命令
3. 每个分组都有完整的开始和结束命令

### 自动复制
1. 勾选"转换后自动复制"
2. 转换完成后会自动复制到剪贴板
3. 可以直接粘贴到CAD软件中使用

### 文件保存
1. 点击"保存文件"按钮
2. 选择保存位置和文件名
3. 保存为TXT格式文件

## ⚠️ 注意事项

### 性能建议
- 坐标点数量建议不超过50,000个
- 大文件处理可能需要较长时间
- 图形预览会限制显示点数以提高性能

### 兼容性
- 支持Windows 7/8/10/11
- 需要Python 3.7或更高版本
- 图形预览需要matplotlib库

### 安全提示
- 程序会自动验证文件路径安全性
- 坐标值会进行合理性检查
- 异常数据会被自动跳过

## 🐛 常见问题

### Q: 程序无法启动
A: 请检查是否安装了Python和依赖包，运行`安装依赖.bat`

### Q: 文件读取失败
A: 检查文件格式和编码，程序支持多种编码格式

### Q: 坐标解析错误
A: 检查坐标格式，确保使用正确的分隔符

### Q: 图形预览不显示
A: 安装matplotlib库：`pip install matplotlib>=3.3.0`

### Q: 复制功能不工作
A: 检查剪贴板权限，尝试手动复制
